## 📌 Featured projects

### 🤖 Local Job Monitoring Agent — RAG + Local LLM

A Python project designed to analyze job offers and compare them with one or more CVs using a local **RAG pipeline** and a local **LLM via Ollama**.

Main features:

- PDF CV extraction with `pdfplumber`
- Local vector indexing with **ChromaDB**
- Semantic search using embeddings
- Job offer analysis with **Ollama**
- Hybrid scoring between CV and job offer
- Persistent memory of already processed offers
- Streamlit dashboards for monitoring and CV Q/A

**Tech stack:** Python, Ollama, ChromaDB, LangChain, LangGraph, Streamlit, pdfplumber
